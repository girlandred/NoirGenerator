# Support

## Common Issues

**"No API key found"** — Run `Noir Commits: Set API Key` from the Command Palette and paste your key from [console.anthropic.com](https://console.anthropic.com).

**"Authentication failed"** — Your API key may be invalid or revoked. Remove it with `Noir Commits: Remove API Key`, then set a new one.

**"Could not read the last commit"** — Make sure the open workspace is a git repository with at least one commit.

**Chapter already exists** — Each commit only generates one chapter. Use `Noir Commits: Skip Commit` if you want to skip the current commit and move on.
